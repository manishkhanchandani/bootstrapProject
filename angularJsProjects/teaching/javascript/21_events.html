JAVASCRIPT EVENTS abort occurs when user cancels loading of an image blur occurs when input focus is removed from a form element click occurs when the user clicks on a link or form element change occurs when the value of form field is changed error occurs when error happens during loading of document focus occurs when input focus is given to a form element load occurs when page is loaded mouseout occurs when user moves the pointer off a link or area mouseover occurs when user moves the pointer over a link or area reset occurs when user clears a form using reset button select occurs when user selects a form elements field submit occurs when a form is submitted unload occurs when the user leaves a page JAVASCRIPT EVENT HANDLERS onclick onmouseover onfocus etc.
